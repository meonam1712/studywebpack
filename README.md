## Installation
--------------------

```sh
### install nodejs
sudo apt-get install nodejs

### install npm.
sudo apt-get install npm

### install git
sudo apt-get install git

### everything is ready.
git clone git@git.lozi.space:web/masamune-desktop.git
cd masamune-desktop

### install everything
npm install

### run project
npm run watch

```
